# CityGMLモデル生成プラグイン

作成されている地図のテーマを編集するQGISプラグインです。<BR>
Shapeファイルや基盤地図(xml)を読み込み、高さを設定して、CityGMLを出力します。 <BR>
出力したCityGMLの内容の検査もします。
